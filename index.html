Why Moxi?

Everything you need. Nothing you don't.

Moxi keeps mod management out of your way. Auto-detect your games, install in one click, and get back to playing.

Auto-Detect

Moxi scans your system and automatically finds every supported game you have installed.

Mod Database

Browse a curated database of mods for each game. Filter, search, and install instantly.

One-Click Install

Install and remove mods without touching a single file manually. Moxi handles everything.

Clean Profiles

Keep your mod lists organized per game. No cross-contamination, no broken installs.

Get Started

Up and running in three steps.

01

Download Moxi

Grab the latest release from GitHub. Moxi is a lightweight desktop app with no heavy runtime requirements.

→
02

Open and Detect

Launch Moxi and it will automatically scan your system for supported games. No manual paths needed.

→
03

Browse and Install

Head to the Mod Database, find something you like, and click Install. That's it. Go play.
